2016-04-01

据说程序员喜欢折腾的几样东西是键盘、编辑器、字体，双手赞成！一直在尝试/找寻轻便、实用、跨平台、功能强大、主题丰富（原谅我要求太多）的编辑器，直到接触了号称最性感编辑器的Sublime，其他编辑器从此变路人。

1. 安装Sublime Text及其插件

到Sublime Text官网下载指定版本(推荐使用压缩包形式，解压即可用)：
http://www.sublimetext.com/

1.1 Linux下破解Sublime Text 2

默认安装的Sublime Text是没有注册的，软件顶部会显示(UNREGISTERED)，但不影响使用。这里介绍Linux下对Sublime Text 2的破解。

  • vim sublime_text
  • 将文件转成十六进制形式，vim输入:%!xxd
  • 查找数字串“4333 3342 3032”，将其中的3342改为3242 (完整查找不一定找到，建议查找4333，再一个一个的看)
  • 将文件转换回去，vim输入:%!xxd -r，保存退出
  • 运行程序，Enter License注册（附两枚算好的License）

手动安装

  • 在菜单栏点击Preferences -> Browse Packages…，此时会打开一个名为Packages的文件夹
  • 退回Packages文件夹的上一级目录，找到一个名为Installed Packages的文件夹
  • 下载Package Control.sublime-package，移动到Installed Packages目录
  • 重启Sublime Text

参考网站：https://packagecontrol.io/installation

1.3 安装/删除/查看插件

Ctrl + Shift + P，弹出编辑框

  • 安装： 输入install，选择Package Control: Install Package
  • 卸载： 输入remove，选择Package Control: Remove Package
  • 查看： 输入list，选择Package Control: List Package
