apt-get命令使用

2016.03.31 14:31:03
software

记录一些apt-get的基本操作和设置。

基本命令

  1. 更新源(/etc/apt/sources.list)

    $ sudo apt-get update

  2. 更新所有已安装的软件包

    $ sudo apt-get upgrade

  3. 升级系统

    $ sudo apt-get dist-upgrade

  4. 安装(升级)一个或多个软件

    $ sudo apt-get install pac1 pac2 pac3…

  5. 删除软件，用法同install，会保留配置文件

    $ sudo apt-get remove pac1 pac2 pac3…

  6. 用法通remove，将删除配置文件

    $ sudo apt-get purge pac1 pac2 pac3…

  7. 更新缓存，检查是否有损坏的依赖

    $ sudo apt-get check

  8. 清除不能下载和无用的包

    $ sudo apt-get autoclean
    $ sudo apt-get autoremove

使用代理

  1. 使用配置文件

    $ cat apt_proxy.conf
    Acquire::http::proxy “http://127.0.0.1:8000/“;
    Acquire::ftp::proxy “ftp://127.0.0.1:8000/“;
    Acquire::https::proxy “https://127.0.0.1:8000/“;
    $ sudo apt-get -c apt_proxy.conf search/install/remove/…

  2. 命令行指定参数

    $ sudo apt-get -o Acquire::http::proxy=”http://127.0.0.1:8000/“ search/install/remove/…

  3. 设置环境变量

    $ export http_proxy=http://127.0.0.1:8000
    $ export https_proxy=http://127.0.0.1:8000
    $ sudo apt-get search/install/remove/…
