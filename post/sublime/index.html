安装Sublime Text及其插件

到Sublime Text官网下载指定版本(推荐使用压缩包形式，解压即可用)： http://www.sublimetext.com/

Linux下破解Sublime Text 2

默认安装的Sublime Text是没有注册的，软件顶部会显示(UNREGISTERED)，但不影响使用。这里介绍Linux下对Sublime Text 2的破解。

  • vim sublime_text
  • 将文件转成十六进制形式，vim输入:%!xxd
  • 查找数字串“4333 3342 3032”，将其中的3342改为3242 (完整查找不一定找到，建议查找4333，再一个一个的看)
  • 将文件转换回去，vim输入:%!xxd -r，保存退出
  • 运行程序，Enter License注册（附两枚算好的License）

安装Package Control

在线安装

  • “Ctrl+`”进入命令行模式，输入以下代码并回车
  • Sublime Text 2:
import urllib2,os,hashlib;
h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88';
pf = 'Package Control.sublime-package';
ipp = sublime.installed_packages_path();
os.makedirs( ipp ) if not os.path.exists(ipp) else None;
urllib2.install_opener( urllib2.build_opener( urllib2.ProxyHandler()) );
by = urllib2.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read();
dh = hashlib.sha256(by).hexdigest();
open( os.path.join( ipp, pf), 'wb' ).write(by) if dh == h else None;
print('Error validating download (got %s instead of %s), please try manual install' % (dh, h) if dh != h else 'Please restart Sublime Text to finish installation')
  • Sublime Text 3:
import urllib.request,os,hashlib;
h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88';
pf = 'Package Control.sublime-package';
ipp = sublime.installed_packages_path();
urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) );
by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read();
dh = hashlib.sha256(by).hexdigest();
print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

手动安装

  • 在菜单栏点击Preferences -> Browse Packages…，此时会打开一个名为Packages的文件夹
  • 退回Packages文件夹的上一级目录，找到一个名为Installed Packages的文件夹
  • 下载Package Control.sublime-package，移动到Installed Packages目录
  • 重启Sublime Text

参考网站： https://packagecontrol.io/installation

安装/删除/查看插件

Ctrl + Shift + P，弹出编辑框

  • 安装： 输入install，选择Package Control: Install Package
  • 卸载： 输入remove，选择Package Control: Remove Package
  • 查看： 输入list，选择Package Control: List Package

配置

Sublime Text基本配置

Preferences -> Settings - User (详细的参数配置可以在Settings - Default中查看)：

{
    "bold_folder_labels": false,
    "ensure_newline_at_eof_on_save": true,
    "draw_white_space": "all",
    "font_size": 11,
    "highlight_line": true,
    "rulers":    // 右边80, 100, 120个字符处显示边界线
    [
        80,
        100,
        120
    ],
    "show_line_endings": true,
    "translate_tabs_to_spaces": true,
    "trim_automatic_white_space": true,
    "trim_trailing_white_space_on_save": false,
    "highlight_modified_tabs": true,
    "word_wrap": "true"
}
