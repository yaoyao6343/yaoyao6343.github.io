Vim配置及插件

Apr 7, 2016 · 1 minute read

记录经常使用的Vim配置和插件。

vim配置文件

适用于当前用户

$ cat ~/.vimrc
set nu
set cursorline
nmap <space> :
colorscheme desert

""" Use Powerline
set laststatus=2
set t_Co=256
set rtp+=/usr/local/lib/python2.7/dist-packages/powerline/bindings/vim

适用于所有用户

$ tail -10 /usr/share/vim/vimrc
if filereadable("/home/yaoyao/.vimrc")
  source /home/yaoyao/.vimrc
endif

Powerline插件

Powerline是使用Python开发的一个外挂小工具，支持各种常见的Shell和应用程序，可以产生非常漂亮的提示字符与状态列文字，让终端文字看起来更舒服。

安装Powerline

到官网下载Powerline源码包，解压后安装：

$ sudo python setup.py install

在Vim上使用

在.vimrc中添加配置：

set laststatus=2
set t_Co=256
set rtp+=/usr/local/lib/python2.7/dist-packages/powerline/bindings/vim
